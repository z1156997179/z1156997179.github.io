小米路由器MT工具箱

发表于 2019-05-19 | 分类于 软件工具 | 阅读次数: |
字数统计: 1.5k 字 | 阅读时长 ≈ 5 分钟

插件名称：Misstar Tools工具箱
插件类型：插件集合工具箱
SSH权限：需要SSH权限
工具箱作者：Starry
讨论QQ群：**① 群-523723125 ② 群-157558789（群内可获得第一手更新信息和插件内测信息，PS：禁止违法话题和小广告） 路由器支持：小米路由器硬盘版1/2（R1D&R2D），小米路由器3（R3），小米路由器mini，小米路由器pro，小米路由器HD，小米路由3G 工具箱版本：**2.7.05.21（支持后续一键更新）

工具箱简介：
MT工具箱是目前第三方插件里面最为方便易用的插件集合，往往我们以前安装各种插件，都要一个个在SSH里面安装并且还一个个设置都是比较麻烦的！MT工具箱给我们提供一个整体的框架，在工具箱里面的每个插件都是独立的。所以我们可以自由选择需要自己安装的插件，并且可以单独更新插件。MT工具箱直接集成在路由器后台页面，只有安装的时候需要一行SSH命令，往后的插件安装卸载升级都只需要在这一个后台页面进行，全图形化操作，特别的方便！有问题的时候不用再经常使用SSH去查看每个插件的运行状况和设置了！并且工具箱是免疫OTA升级的，可以放心更新系统，工具箱依然坚挺的存在！

插件介绍：
MT工具箱目前提供的插件还是比较丰富的，基本能满足需求。包括如下插件：KMS服务器，VSFTP服务器，VPN服务器，远程管理，ARIA2，Koolproxy广告过滤，阿呆喵广告过滤，SS（由于不可描述的原因暂时下架），webshell，aliddns，frp内网穿透。
当然后续会提供更多的插件，直接在页面的插件中心安装即可。

VPN服务器：
这个插件可以让路由建立一个VPN服务器，用于外网连接路由器，可以用于读取路由内网共享资源，一些只有内网能操作的东西，可以联机玩游戏等等，这个可以自行搜索下VPN！由于R1D有些问题没有这个插件所以这里没有相关图片之类的提供！有兴趣的可以安装工具箱！
KMS服务器：
KMS大家应该都不陌生，没错，我们现在激活工具用的最多的就是KMS软件了！不过大家都这道，网上这类的工具比较乱，很多都附带恶意软件！这个插件是直接把路由器变成一个KMS服务器，支持内网和外网（配合DDNS）使用，十分方便！以后激活就靠自己就好了！

VSFTP服务器：
之前论坛上有朋友发过VSFTPD这个插件了，本人也转载过一份，但是安装和配置起来都比较麻烦！现在MT工具箱把这个工具图形化了，设置很方便，同时也支持多用户登录不同文件夹，满足对于一个基本FTP服务器的需求！配合DDNS还可以开启外网访问，算是一个简易云存储了！

远程管理&webshell：
这个插件主要是提供远程访问的端口设置，支持远程SSH登录和web管理后台登录。方便远程设置的操作需求
webshell则是提供一个网页端的SSH终端

广告过滤（ADM广告过滤&koolproxy广告过滤）：
MT工具箱提供了两款广告过滤工具，ADM很多人应该都不陌生了，但是路由版的ADM效果一般并且十分耗费资源所以不推荐使用。好在最近作者添加了KP广告过滤插件，KP也很出名，移植自梅林固件同名插件，去广告的效果十分赞，占用资源也低，还支持每台机器单独配置，支持https，主程序和规则也可以自动更新，个人十分推荐这个插件！

工具箱安装：
说了这么多，这个工具箱到底怎么安装？其实很简单，
首先确定你的路由器是R1D,R2D,R3,mini,R3P,R3D这几个的其中一个
其次你需要刷开发版并且开启SSH，SSH的开启和使用论坛已经有太多相关帖子，这里就不多说了！附上官方SSH链接https://d.miwifi.com/rom/ssh （注意，开启SSH意味着放弃保修）
最后用软件登上SSH（putty xshell bash随便啦）输入下面这一段安装命令，按照说明操作即可，整个过程不用1分钟。安装完直接登录后台管理页面就可以看到MT工具箱了，然后之后就不用ssh工具了！

安装：

wget http://www.misstar.com/tools/appstore/install.sh -O /tmp/install.sh && chmod +x /tmp/install.sh && /tmp/install.sh

（已安装的1.1.0或者1.3.0可以网页里面直接点升级，但是版本升级会都是配置信息，如有重要数据请做好备份）
卸载：也可直接在管理页面卸载

wget http://www.misstar.com/tools/uninstall.sh -O /tmp/uninstall.sh && chmod +x /tmp/uninstall.sh && /tmp/uninstall.sh

MT工具箱的优点有很多，集成到管理后台，免疫OTA更新，可以放心升级官方固件，工具箱不会丢（除非桶reset或者U盘刷机）！插件安装升级以及工具箱本身的升级都十分方便快捷！插件也满足了大部分人的需求，该有的都有了，并且后续还会添加其他插件，有这三款路由的朋友绝对不能错过！
